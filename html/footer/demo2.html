最流行的浏览器。

Google Chrome

Google Chrome 是由 Google 开发的网络浏览器，于 2008 年发布。Chrome 是当今世界上最流行的网络浏览器！

Mozilla Firefox

Mozilla Firefox 是 Mozilla 开发的开源网络浏览器。自 2018 年 1 月以来，Firefox 一直是第二受欢迎的网络浏览器。

Microsoft Edge

Microsoft Edge 是 Microsoft 开发的网页浏览器，于 2015 年发布。Microsoft Edge 取代了 Internet Explorer。
